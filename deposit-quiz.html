1📋 เปิดบัญชี
การแสดงตนของกรรมการผู้มีอำนาจและผู้ถือหุ้น ในกรณีนิติบุคคลประเภทบริษัทจำกัด จะต้องตรวจสอบเอกสารแสดงตนของผู้ถือหุ้นที่มีสัดส่วนการถือหุ้นตั้งแต่เท่าใดขึ้นไป สำหรับสัญชาติไทย?
คำตอบ: D
ตามระเบียบการเปิดบัญชีลูกค้านิติบุคคล ต้องตรวจสอบเอกสารแสดงตน (บัตร ปชช./Passport) ของกรรมการผู้มีอำนาจลงนาม และผู้ถือหุ้นสัญชาติไทยตั้งแต่ 15%ขึ้นไป ส่วนสัญชาติอื่นตั้งแต่ 10% ขึ้นไป (อ้างอิงหน้า 4)
2📋 เปิดบัญชี
การเปิดบัญชีเงินฝากของวัด ตามมติมหาเถรสมาคม ครั้งที่ 4/2569 — รหัสประเภทลูกค้าสำหรับบัญชีวัดคือข้อใด?
คำตอบ: A
มติมหาเถรสมาคม ครั้งที่ 4-2569 (ลง 28 พ.ค.69) กำหนดให้ใช้รหัสประเภทลูกค้า 0702 สำหรับบัญชีวัด (สถาบันที่ไม่มีวัตถุประสงค์เพื่อหากำไร) — ใช้คำว่า "วัด" นำหน้าชื่อ · 1 วัด = 1 CIF
3🏥 ถอนผู้ป่วย
ในกรณีผู้ฝากเงินป่วยและไม่มีสติสัมปชัญญะสมบูรณ์ โดยศาลยังไม่มีคำสั่งใดๆ (บุคคลวิกลจริต) — หากในขณะที่พนักงานไปพบลูกค้า สามารถพูดคุยรู้เรื่องและมีสติในขณะนั้น พนักงานควรปฏิบัติอย่างไร?
คำตอบ: C
บุคคลวิกลจริต (ศาลยังไม่สั่ง) → หากขณะติดต่อมีสติสัมปชัญญะครู่หนึ่ง ลูกค้าถอนเงินเองได้ (อ้างอิงหน้า 10 — หลักการคือสถานะทางกฎหมาย ไม่ใช่อาการป่วย)
4🌐 ต่างสำนักงาน
เงื่อนไขการให้บริการเกี่ยวกับเงินฝากต่างสำนักงาน สำหรับบัญชีกระแสรายวันของ "นิติบุคคล" มีข้อจำกัดอย่างไร?
คำตอบ: A
บัญชีกระแสรายวันของนิติบุคคล — ต้องนำฝากเข้าบัญชีตามชื่อหน้าเช็คเท่านั้น จ่ายเงินสดไม่ได้ (ต่างจากบุคคลธรรมดาที่เช็ค ≤ 1 ล้านจ่ายเงินสดได้) (อ้างอิงหน้า 14)
5🔒 อายัด
หน่วยงานใดที่มีอำนาจ "ยึด/อายัดทรัพย์สิน" แต่ "ไม่มีอำนาจ" ขอข้อมูลบัญชีเงินฝาก (ตาม พ.ร.บ.ข้อมูลข่าวสารของราชการ พ.ศ.2540)?
คำตอบ: D
จาก 21 หน่วยงาน — มีเพียงสำนักงานบังคับคดีที่มีอำนาจเฉพาะ "ยึด/อายัด" แต่ไม่มีอำนาจ "ขอข้อมูลบัญชี" (อ้างอิงหน้า 23-24)
6🕊️ จ่ายคืน
กรณีพระภิกษุมรณภาพ เงินฝากที่ได้มา "ขณะที่อยู่ในสมณเพศ" จะถูกจัดการอย่างไรตามระเบียบธนาคาร?
คำตอบ: A
เงินฝากที่ได้มาขณะอยู่ในสมณเพศ = สมบัติของวัด (ตามกฎหมายคณะสงฆ์) — ยกเว้นมีพินัยกรรมระบุไว้ ให้จ่ายตามพินัยกรรม (อ้างอิงหน้า 36)
7🏥 ถอนผู้ป่วย
กรณีผู้ฝากเงินป่วยแต่ "มีสติสัมปชัญญะบริบูรณ์" — พนักงานที่ออกไปให้บริการณ์ที่พัก ต้องพิสูจน์ข้อเท็จจริงว่าอย่างไร?
คำตอบ: C
พนักงานอย่างน้อย 2 คนออกไปให้บริการ ณ ที่พัก/โรงพยาบาล พร้อมพิสูจน์ข้อเท็จจริงว่าป่วยจนมาสาขาไม่ได้ — ใช้ใบถอนเงินแบบ 302.16 + ลายพิมพ์นิ้วมือ + พยานเชื่อถือได้ 2 คน (เช่น แพทย์/พยาบาล)
8🏥 ถอนผู้ป่วย
กรณีศาลสั่งให้ลูกค้าเป็น "บุคคลไร้ความสามารถ" — ผู้ที่มีอำนาจดำเนินการถอนเงินฝากแทนคือใคร?
คำตอบ: C
บุคคลไร้ความสามารถ (ศาลสั่ง) → ผู้อนุบาลดำเนินการแทนเท่านั้น · ส่วนบุคคลเสมือนไร้ความสามารถ (ศาลสั่ง) → ผู้พิทักษ์ (ถ้าศาลสั่งให้ทำแทน) — ใช้แบบ 302.16.02 + คำสั่งศาลแต่งตั้ง
9🏥 ถอนผู้ป่วย
เอกสารที่ต้องใช้ในการถอนเงินฝากของผู้ป่วย (ไม่มีสติสัมปชัญญะ) โดยผู้พิทักษ์/ผู้อนุบาล ต้องใช้แบบพิมพ์ใด และทำที่สาขาใด?
คำตอบ: B
ใช้แบบพิมพ์302.16.02 (คำขอรับเงินผู้ฝากป่วยและไม่มีสติสัมปชัญญะสมบูรณ์) + ติดต่อขอถอนเงิน ณ สาขาที่เปิดบัญชีเท่านั้น — เอกสารครบ 5 รายการ (ศาลสั่ง + สำเร็จ + สมุด + บัตร ผู้กระทำการแทน + บัตร ผู้ป่วย)
10🌐 ต่างสำนักงาน
วงเงินการถอนเงินฝากออมทรัพย์/ประจำ ต่างสำนักงาน ไม่เกินเท่าใดต่อบัญชีต่อวัน (รวมทุกรหัสรายการ ยกเว้นถอนผ่าน ATM และ BAAC Mobile)?
คำตอบ: B
ออมทรัพย์/ประจำ ถอนต่างสำนักงานได้ไม่เกิน1,000,000 บาท/บัญชี/วัน (รวมทุกรหัสรายการ ยกเว้น ATM/BAAC Mobile)
11🌐 ต่างสำนักงาน
ข้อใด "ไม่ใช่" ข้อห้ามในการทำรายการต่างสำนักงาน?
คำตอบ: C
ข้อห้าม 7 ข้อ ได้แก่ ① เปิดบัญชี ② ปิดบัญชี ③ เปลี่ยนแปลงข้อมูล ④ ออกสมุดเล่มใหม่ ⑤ มอบอำนาจถอน ⑥ มอบอำนาจขอ Statement/รับรอง ⑦ แก้ไขรายการข้ามวัน — ฝากเงินสดทำได้ตามปกติ
12🌐 ต่างสำนักงาน
กรณีที่สาขาต่างสำนักงาน "ทำรายการถอนเป็นฝาก" ผิดพลาด (และข้ามวัน/ติดต่อลูกค้าไม่ได้) — ต้องปฏิบัติอย่างไร?
คำตอบ: C
① สาขาที่ทำรายการส่งบันทึกผ่านระบบสารบรรณ → ② สาขาเจ้าของบัญชีอายัดเงิน → ③ ติดต่อลูกค้าให้นำสมุดมาแก้ไข (ถ้าข้ามวันจะใช้ EC ไม่ได้)
13🌐 ต่างสำนักงาน
การถอนสลากออมทรัพย์ต่างสาขา กรณี "เจ้าของสลากมาถอนด้วยตนเอง" ถอนเป็นเงินสดได้ไม่เกินเท่าใดต่อฉบับ?
คำตอบ: C
เจ้าของสลากมาถอนเอง ถอนเงินสดได้ไม่เกิน300,000 บาท/ฉบับ · เกิน → โอนเข้าบัญชีเจ้าของหรือแคชเชียร์เช็ค A/C Payee Only (ชื่อตรงเจ้าของสลาก)
14🌐 ต่างสำนักงาน
บัญชีเงินฝากออมทรัพย์จะถูกเปลี่ยนสถานะเป็น "Unclaimed" เมื่อใด?
คำตอบ: C
ระบบเปลี่ยนสถานะ Active → Unclaimed เมื่อบัญชีออมทรัพย์ไม่เคลื่อนไหวนานกว่า 36 เดือน และมียอดคงเหลือตั้งแต่ 50,000 บาทขึ้นไป (หมายเหตุ: รายการจ่ายดอกเบี้ยทบต้นไม่ถือเป็นการเคลื่อนไหว)
15🌐 ต่างสำนักงาน
ลูกค้ามาติดต่อที่สาขาเพื่อทำธุรกรรม แต่บัญชีอยู่ในสถานะ Unclaimed พนักงานต้องปฏิบัติอย่างไร?
คำตอบ: C
พนักงานผู้รับมอบอำนาจต้องเปลี่ยนสถานะจาก Unclaimed เป็น Active ก่อน จึงจะทำธุรกรรมตามปกติได้ (ระบบจะแสดง Restriction ที่หน้าจอเมื่อบัญชี Unclaimed)
16🔒 อายัด
การอายัดบัญชีเงินฝาก — กรณี "ยอดเงินในบัญชีน้อยกว่าค่าสั่งอายัด" ต้องดำเนินการอย่างไร?
คำตอบ: C
กรณียอดเงิน < ค่าสั่งอายัด → หักเงินเท่าที่เหลือทั้งหมด + ส่งแคชเชียร์เช็ค A/C Payee Only · ถ้ามีเงินโอนเข้าภายหลัง → แจ้งหน่วยงานดำเนินการต่อ
17🔒 อายัด
การอายัด "สลากออมทรัพย์ที่เป็นหลักประกันเงินกู้กับธนาคาร" — ธนาคารต้องดำเนินการอย่างไร?
คำตอบ: B
สลากเป็นหลักประกันกับธนาคาร → ธนาคารมีสิทธิได้รับชำระหนี้ก่อน · อายัดสลากไว้แต่ยังไม่นำส่งเงินจนกว่าจะพ้นภาระ + แจ้งเหตุผลให้หน่วยงานทราบ
18🔒 อายัด
ขั้นตอนการ "ยกเลิกการอายัด" — ขั้นตอนที่ 3 ("สอบทาน") กำหนดให้ใครเป็นผู้สอบทาน และต้องไม่เป็นใคร?
คำตอบ: B
ขั้นที่ 3 สอบทาน ทำโดยพนักงานชั้น ก (คนที่ 2) ที่ไม่ใช่ผู้บันทึกในขั้นตอนที่ 2 (หลักการ "สี่ตา" ป้องกันคนเดียวทำครบทุกขั้น)
19🔒 อายัด
ข้อควรระวังในการอายัด — ข้อใด "ถูกต้อง"?
คำตอบ: B
ข้อควรระวัง 3 ข้อ: ① ห้ามปิดบัญชีที่ถูกอายัด ② ห้ามลูกค้าเคลื่อนไหวบัญชี (โดยเฉพาะถอน) ③ ห้ามปลดการอายัดใน WebCSR โดยไม่มีหนังสือจากหน่วยงานที่มีอำนาจ
20🔒 อายัด
หน่วยงานในข้อใดต่อไปนี้ที่ "มีอำนาจเฉพาะขอข้อมูลบัญชี" เท่านั้น (ไม่มีอำนาจยึด/อายัด)?
คำตอบ: C
จาก 21 หน่วยงาน — มี 2 หน่วยงานที่มีอำนาจเฉพาะ "ขอข้อมูลบัญชี" ได้แก่ สำนักงานตรวจคนเข้าเมือง (สตม.) และส่วนราชการ (สอบสวนวินัยอย่างร้ายแรง) · ส่วน DSI/สำนักงานบังคับคดี/กรมสรรพากรมีอำนาจยึด/อายัดด้วย
21🚪 ปิดบัญชี
การปิดบัญชีเงินฝากมีกี่ประเภท และต้องทำผ่านระบบใดเท่านั้น?
คำตอบ: B
การปิดบัญชีมี 5 ประเภท: เจ้าของ / มอบอำนาจ / ธนาคาร / ศาล / ผู้อนุบาล · ต้องทำผ่านระบบ PDTeller เท่านั้น · ห้ามแก้ไขสถานะเป็น "Closed" ใน WebCSR โดยเด็ดขาด
22🚪 ปิดบัญชี
หนังสือมอบอำนาจในการปิดบัญชี ต้องปิดอากรแสตมป์เท่าใด หาก "กระทำการมากกว่า 1 ครั้ง"?
คำตอบ: C
ปิดอากรแสตมป์ตามประมวลรัษฎากร: 10 บาท = กระทำการครั้งเดียว · 30 บาท = กระทำการมากกว่า 1 ครั้ง
23🚪 ปิดบัญชี
การปิดบัญชี "โดยธนาคาร" — ขั้นตอนที่ 2 (การแจ้งเตือนล่วงหน้า) ต้องส่งหนังสือขอความร่วมมือล่วงหน้าอย่างน้อยกี่วัน?
คำตอบ: C
ขั้นที่ 2 การแจ้งเตือนล่วงหน้า ส่งหนังสือขอความร่วมมือฯ ล่วงหน้า อย่างน้อย 30 วัน (ไปรษณีย์ลงทะเบียนตอบรับ)
24🚪 ปิดบัญชี
หากลูกค้าไม่มาติดต่อตามแจ้งเตือน — ธนาคารปิดบัญชีและโอนเงินเข้าบัญชีใด (ใช้รหัสบัญชีอะไร)?
คำตอบ: C
หากลูกค้าไม่มาติดต่อ → ปิดบัญชี + พิมพ์ Statement + โอนเงินเข้า "บัญชีเจ้าหนี้รอจ่ายคืนลูกค้า" (รหัส 23507007) + ส่งหนังสือแจ้งปิดบัญชี
25🚪 ปิดบัญชี
กรณีลูกค้าถูกศาลสั่งพิทักษ์ทรัพย์ และต้องการปิดบัญชี — เอกสารที่ต้องใช้คือข้อใด?
คำตอบ: B
กรณีถูกศาลสั่งพิทักษ์ทรัพย์ / บุคคลล้มละลาย / เลิกกิจการ → ใช้แบบหนังสือคำสั่งศาล หรือหนังสือแจ้งตั้งผู้ชำระบัญชี และดำเนินการตามขั้นตอน (ผู้มีอำนาจอนุมัติ = พนักงานผู้รับมอบอำนาจ ชั้น ก)
26🕊️ จ่ายคืน
กรณีลูกค้า "ล้มละลาย" — ผู้มีสิทธิขอรับเงินฝากแทนเป็นใคร?
คำตอบ: B
ลูกค้าล้มละลาย → เจ้าพนักงานพิทักษ์ทรัพย์เป็นผู้มีอำนาจขอรับเงินฝากแทน เท่านั้น (ต่างจากกรณีตายที่ทายาท/ผู้จัดการมรดก/ลูกค้าร่วมมีสิทธิ)
27🕊️ จ่ายคืน
อำนาจอนุมัติจ่ายเงินให้ทายาทที่เกิน 1,000,000 บาท — เงื่อนไขคืออะไร และผู้อนุมัติระดับใด?
คำตอบ: B
อนุมัติจ่ายทายาทโดยธรรม: เกิน 1,000,000 บาท → ต้องใช้คำสั่งศาลแต่งตั้งผู้จัดการมรดกเท่านั้น · ผู้อนุมัติ = พนักงานผู้รับมอบอำนาจ ชั้น ก (จำง่าย: วงเงินเกิน 1 ล้าน ใช้ศาลอย่างเดียว)
28🕊️ จ่ายคืน
กรณีบัญชีร่วมที่ผู้ฝากร่วม "ล้มละลาย" — ใครต้องลงลายมือชื่อร่วมกันในการถอนเงินเพื่อปิดบัญชี?
คำตอบ: C
บัญชีร่วมล้มละลาย → เจ้าพนักงานพิทักษ์ทรัพย์ (ตัวแทนผู้ล้มละลาย) และลูกค้าร่วมคนอื่นๆ ต้องลงลายมือชื่อร่วมกัน (ไม่ใช่ผู้จัดการมรดก เพราะล้มละลายไม่ใช่ตาย)
29🕊️ จ่ายคืน
กรณีลูกค้าชาวต่างชาติ (มีถิ่นพำนักนอกประเทศ) เสียชีวิต และไม่สามารถปฏิบัติตามระเบียบธนาคารฉบับที่ 14 ได้ — สาขาต้องปฏิบัติอย่างไร?
คำตอบ: C
กรณีไม่สามารถปฏิบัติตามระเบียบฯ ได้ → ให้สาขาทำบันทึกหารือมายังสำนักงานใหญ่ เพื่อพิจารณาให้ความเห็นชอบเป็นราย ๆ ไป
30🕊️ จ่ายคืน
กรณีพระภิกษุสึกจากสมณเพศ — เงินฝากที่ฝากไว้ "ก่อนอุปสมบท" จะถูกจ่ายคืนให้ใคร?
คำตอบ: B
กรณีฝากก่อนอุปสมบท เงินนั้นเป็นทรัพย์สินส่วนบุคคล (ไม่ใช่สมบัติของวัด) → จ่ายคืนทายาท · ส่วนที่ฝากหลังอุปสมบท (ขณะเป็นพระ) = สมบัติของวัด
