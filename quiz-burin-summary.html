🧠 2. หลัก 7C's + 5P's (กรอบวิเคราะห์สินเชื่อ)

📌 7C's (สหกรณ์รายใหญ่)

C1: Character — ความประพฤติ/ชื่อเสียง
C2: Capacity — ความสามารถชำระหนี้ (วิเคราะห์ Cash Flow)
C3: Capital — เงินทุน/โครงสร้างทางการเงิน (D/E Ratio)
C4: Collateral — หลักประกัน (LTV)
C5: Condition — สภาพเศรษฐกิจ/อุตสาหกรรม
C6: Capital Source — แหล่งเงินทุน
🎯 C7: Control — ระบบการควบคุมภายใน + ความโปร่งใส + กลไกการตรวจสอบ (เช่น กรมตรวจบัญชีสหกรณ์) — หัวใจของลูกหนี้สหกรณ์

📌 5P's (วิเคราะห์ Payment - ความสามารถชำระหนี้)

P1: People — ผู้กู้ (ประวัติ วินัยทางการเงิน)
P2: Purpose — วัตถุประสงค์เงินกู้
🎯 P3: Payment (Primary Source) — กระแสเงินสดจากการดำเนินงานปกติ (Cash Flow from Core Business) เสมอ — ไม่ใช่จากยึดทรัพย์/ประกัน
P4: Protection — หลักประกัน/Secondary Source
P5: Perspective — มุมมองระยะยาว
⚠️ Payment (P3) ต้องมาจาก "กระแสเงินสดจากการดำเนินงานปกติ" เช่น รายได้จากการขายข้าวตามสัญญา Forward Contract — ไม่ใช่เงินจากการยึดทรัพย์/เคลมประกัน

📊 3. Credit Risk Rating — เกณฑ์คะแนนและอัตราดอกเบี้ย (ฉ.26)

3.1 โครงสร้าง 3 หมวด รวม 100 คะแนน

หมวด 1: ความมั่นคง (Stability)
หมวด 2: ความสามารถในการดำเนินธุรกิจสินเชื่อของสหกรณ์ (Business Capability)
หมวด 3: ประวัติการชำระหนี้ (Repayment History)
รวม: 100 คะแนน → ยิ่งสูงยิ่งดี

3.2 อัตราดอกเบี้ยตามระดับคะแนน (กรณีทั่วไป)

เกณฑ์คะแนนระดับอัตราดอกเบี้ย
ระดับ 7≥ 60 คะแนนMLR + 1.50
ระดับ 8 (≤ 57)คะแนน ≤ 57MLR + 1.75
ระดับ 8 (≥ 53)คะแนน ≥ 53MLR + 1.75
ระดับ 9< 53MLR + 2.00
กรณี ODเพิ่มชั้นความเสี่ยง+ 0.25 ถึง + 3.00
🎯 กฎ Floor: ลูกค้าเดิมดำเนินงาน < 2 ปีบัญชี → ต้องไม่ต่ำกว่าระดับ 7 (MLR+1.50) ถึงแม้คะแนนจะดีกว่า

3.3 ลักษณะการกำหนดดอกเบี้ย

กำหนดเป็นรายลูกค้า (Customer Based)
  • MLR + ชั้นความเสี่ยง 0.25-2.00
  • กรณี OD + ชั้นความเสี่ยง 0.25-2.00
ห้าม!
  • ❌ กำหนดเป็นรายสัญญา (รายเดียวกัน หลายสัญญา)
  • ❌ OD ใช้ชั้นความเสี่ยง 0.25-3.00

📈 4. อัตราส่วนทางการเงิน 6 ตัว (ลูกค้ารายใหญ่)

อัตราส่วนสูตรเกณฑ์
1. LTV จำนวนหนี้เงินกู้ × 100 / มูลค่าหลักประกันจำนอง ค่าต่ำดี — หลักประกันสูงคุ้มวงเงิน
2. D/E Ratio หนี้สินรวม / ส่วนของเจ้าของ ค่าสูงไม่ดี — กู้ยืมสูง vs ทุนตัวเอง
3. Gross Profit Margin (ยอดขาย - ต้นทุนขาย) × 100 / ยอดขาย ค่าสูงดี
4. ICR (Interest Coverage) EBITDA / ดอกเบี้ยจ่าย ค่าสูงดี
5. DSCR (Debt Service Coverage) EBITDA / (ดอกเบี้ยจ่าย + ต้นเงินกู้ระยะยาวถึงกำหนดในปี) ค่าสูงดี
EBITDA = กำไรดำเนินงาน + ค่าเสื่อมราคา + ค่าตัดจำหน่าย

📌 เกณฑ์การลงทุน (โครงการลงทุนระยะยาว)

ตัววัดเกณฑ์ตัดสิน
6.1 NPV (มูลค่าปัจจุบันสุทธิ)NPV ≥ 0 — ผลตอบแทนคุ้มค่าเสียโอกาส
6.2 IRR (อัตราผลตอบแทน)IRR > อัตราดอกเบี้ยเงินกู้ — กิจการมีกำไรเหมาะสมลงทุน
6.3 Pay Back periodการชำระคืนต้นเงินกู้ ไม่เกิน 1.5-2.0 เท่า จากผลการคำนวณ
6.4 B/C Ratioค่า > 1 — ผลตอบแทน > ค่าใช้จ่าย

💰 5. วงเงิน O/D (เบิกเงินเกินบัญชี) — สูตร 40%

📌 ตามคู่มือ OD ข้อ 202.06 — ยกเว้นสหกรณ์และองค์กร

5.1 หลักการ

WCR = ลูกหนี้การค้า + สต็อกสินค้า/วัตถุดิบ − เจ้าหนี้การค้า
วงเงิน OD = WCR ที่วิเคราะห์ได้ − วงเงิน W/C ที่มีกับ ธ.ก.ส. − วงเงิน W/C ที่มีกับสถาบันการเงินอื่น
⚠️ ต้องไม่เกิน 40% ของ W/C need (ยกเว้นสหกรณ์/องค์กร)

5.2 ตัวอย่างการคำนวณ

โจทย์: W/C = 90 ล้าน (40% = 36 ล้าน)
รายการวงเงิน (ล้านบาท)
วงเงินกู้เครดิตเงินสด ธ.ก.ส.10.00
วงเงินตั๋วสัญญาใช้เงิน (P/N)15.00
วงเงิน O/D ธ.พาณิชย์25.00
คำนวณ OD90 − 10 − 15 − 25 = 40 ล้าน
ถูกบังคับให้ ≤ 36 ล้าน (หลัก 40%)วงเงิน OD ที่ได้ = 36 ล้าน

5.3 สูตร WCR แบบสัดส่วนเวลา (สหกรณ์ซื้อเงินสด ขายเชื่อ 180 วัน เก็บสต๊อค 90 วัน)

WCR = (ยอดขายเชื่อ × 180 / 360) + (ต้นทุน × 90 / 360) − 0
WCR สูงสุด ≤ 40% ของ WCR need (ยกเว้นสหกรณ์)

📬 7. การเรียกคืนเงินกู้ + บอกกล่าวผู้ค้ำประกัน

7.1 เรียกคืนเงินกู้ — ขั้นตอน WebCSR

ขั้นตอนรายละเอียด
1. ส่งหนังสือเรียกคืนผจก.สาขา ส่งแจ้งสถาบันผู้กู้
2. ครบ 30 วันหัวหน้าหน่วยอำเภอ บันทึกเปลี่ยนสถานะใน WebCSR
3. พิมพ์รายงานZSCA 359A – Daily File Maintenance
4. เสนอ ผบ.ตรวจสอบความถูกต้อง
🚨 กรณีเร่งด่วน: สาขาเสนอรายงานเรียกคืนผ่าน ผอ.ฝสข. ได้ทันที โดยไม่ต้องรอสอบทานหนี้/ตรวจสอบทรัพย์สินเสร็จ — แต่ต้องชี้แจงเหตุผลความจำเป็น

7.2 บอกกล่าวผู้ค้ำประกัน (สัญญาต้นเงินระยะยาว ฉ.52)

กรณีการบอกกล่าว
กรณีที่ 1: สัญญาเงินกู้ทุนหมุนเวียนรายงวด + ผิดนัด + ไม่ได้รับผ่อนผัน • บอกกล่าวภายใน 60 วันนับจากผิดนัด (รอบแรก)
• เมื่อครบตารางชำระ + ยังผิดนัด + ไม่ได้รับผ่อนผัน → บอกกล่าวอีกครั้งภายใน 60 วัน
• ผ่อนผันแล้ว = ไม่ต้องบอกกล่าวในปีนั้น
กรณีที่ 2: สัญญาเงินกู้ลงทุนรายงวด + ผิดนัด • บอกกล่าวภายใน 60 วันนับจากผิดนัดในแต่ละงวด
• งวดต่อไปผิดนัดอีก → บอกกล่าวภายใน 60 วันเช่นกัน
• ทำซ้ำจนกว่าจะชำระเสร็จ

7.3 ปัจจัยที่ไม่พิจารณาดำเนินคดีล้มละลาย
